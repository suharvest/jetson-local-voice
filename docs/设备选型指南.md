# 边缘AI音频服务设备选型指南

> **核心优势**：音频 AI 比视觉 AI 更易落地 —— Jetson 4012 已足够，延迟 <1s，适配工作小（一次声纹注册，无需模型训练）

---

## 市场机会

### 音频 vs 视觉落地难度

| 维度 | 视觉 AI | 音频 AI |
|------|---------|---------|
| 定制成本 | 多模型训练（检测+识别+追踪） | **一次声纹注册** |
| 数据采集 | 数周标注数据集 | **几分钟录音** |
| 部署周期 | 2-4周 | **1-2天** |

### 核心场景

智能客服、语音助手、会议转录、多语言交互、隐私敏感场景（医院/银行/政务）

## Jetson Orin NX/Nano

### Qwen3-TTS + Qwen3-ASR（TRT）

**实测性能（Orin NX 16GB）**：

| 阶段 | 耗时 | 加速方式 |
|------|------|----------|
| Prefill | **15ms** | TRT BF16 |
| CP/step | **5ms** | TRT BF16 |
| Vocoder | **98ms** | TRT FP16 |
| Decoder/step | **21ms** | TRT BF16 |

**优化效果**：

| 指标 | 优化前 | 优化后 | 提升 |
|------|--------|--------|------|
| Prefill | 200ms | 15ms | 13x |
| CP/step | 45ms | 5ms | 9x |
| Vocoder | 465ms | 98ms | 4.7x |
| 内存 | 14GB | 10GB | -4GB |

### Paraformer + Matcha（sherpa）

**实测性能**：

| 指标 | 值 |
|------|-----|
| RTF | **1.9** |

## 销售选型

### 按场景

| 场景 | 推荐 | 原因 |
|------|------|------|
| 高并发 V2V | Jetson NX + Qwen3 TRT | RTF 0.33/0.205，10并发 |
| 标准交互 | Jetson Nano + Qwen3 TRT | 延迟适中，$250-400 |
| 成本敏感 ASR | RK3576 + Qwen3-ASR | RTF=0.44，$150-200 |
| 纯转录 | 树莓派 + SenseVoice | $80-100 |

### 避坑

| 不推荐 | 原因 |
|--------|------|
| RK3576 TTS+ASR 共存 | 8GB OOM |
| RK3576 高频 TTS | RTF=1.9，延迟高 |
| 树莓派实时交互 | 无加速硬件 |

## 技术踩坑

### Jetson TRT

| 问题 | 解决 |
|------|------|
| FP16 NaN（attention溢出） | **BF16** |
| TRT shape 动态 SIGSEGV | **dual profile** |
| EOS 不触发 | **progressive bias** |
| CP CER 30%（并行vs串行） | **autoregressive** |
| ORT CUDA EP 污染 | **独立 Ort::Env** |

### RK3576 RKLLM

| 问题 | 解决 |
|------|------|
| RTF>10（FP16 RKNN慢） | **RKLLM W4A16** |
| EOS 不触发 | **keep_history=1** |
| Vocoder 10s（Sin CPU fallback） | **Taylor 多项式** |
| Encoder INT8 质量崩 | **保持 FP16** |
