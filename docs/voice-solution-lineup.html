档次速记：
前端（ESP32S3 ~$5）— 低功耗唤醒词检测，搭配上位机使用
入门级（reRouter / CM4 $200-300）— 纯转录，~5 种语言，无合成，适合简单记录场景
主流级（RK3576 $200+ / RK3588 $350+）— ASR 52+ 语言 + 轻量 TTS（17 种语言可选但一次只能用一种），音色机械、无克隆
专业级（Orin Nano $600+ / Orin NX $800+）— 52+ 语言 ASR + Qwen3-TTS（待验证）+ 声音克隆
旗舰级（AGX Orin ~$2000）— 全部语音能力 + 本地 LLM，语音与大模型同机部署

选型建议：只需唤醒 → ESP32S3；只需转录 → reRouter；需要双向语音交互 → RK 系列；需要克隆或高品质多语言 → Orin Nano/NX；需要语音 + LLM 一体 → AGX Orin
